# 🚀 Mitra Hasanah - Frontend User Portal v1.0

Portal utama **Mitra Hasanah** yang dibangun dengan performa tinggi dan sinkronisasi realtime. Bukan sekadar landing page statis, ini adalah mesin bisnis yang terintegrasi penuh dengan Dashboard Admin.

## 🛠️ The Wizard's Tech Stack
- **Framework:** React + Vite (Kenceng, kaga pake loading lama)
- **State Management:** Zustand (Global store buat sinkronisasi data tanpa pusing)
- **API Client:** Axios (Terkoneksi ke Backend Wizard)
- **Styling:** Tailwind CSS (Modern, Clean, Dark-mode ready)
- **SEO & Meta:** React Helmet Async (Biar kalau di-share di WA, muncul preview gahar)
- **Icons:** Lucide React (Icon set yang tajam dan type-safe)

## ✨ Fitur "Wizard" yang Udah Implement
- **Realtime Integration:** Data Produk, Baitul Maal, dan Galeri ditarik langsung dari API. Tambah di Admin, muncul di sini detik itu juga.
- **Dynamic SEO:** Setiap halaman punya metadata unik buat kebutuhan sharing sosmed.
- **Type-Safe Component:** Gak ada tempat buat `any` di sini. Semua data ter-interface dengan ketat.
- **Responsive & Interactive:** Cakep di Axioo, ganteng di iPhone.

## 🏃‍♂️ Cara Running (Local Mode)
Pastiin **Bun** udah terinstal di mesin lu biar kenceng.
